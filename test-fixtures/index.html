Detectors C & B — real files (drop into files/)

Negative cases — content that isn’t GPS

Each file carries a GPS extension but holds non-GPS content. Clicking a link opens the dialog (Detector C flags by extension) — Send it and CTM rejects the content, so the dialog should show CTM’s actual error. The fetch button next to each exercises Detector A, which inspects the body and must do nothing (no dialog).

  • webpage.gpx — an HTML page named .gpx.
  • styled-blob.gpx — JSON that quotes <gpx> text (GitHub /_styled shape); the structural sniff must not be fooled.
  • notes.tcx — XML whose root is <notes>, not TrainingCenterDatabase.
  • empty.kml — plain text named .kml.
  • fake.kmz — plain text named .kmz (no zip PK signature).
